# Jekyll docs
https://jekyllrb.com/docs/

## Local build
To serve locally: bundle exec jekyll serve
Go to: http://localhost:4000/
To stop: CTRL + C

To build: bundle exec jekyll build

## New post
https://jekyllrb.com/docs/posts/

bundle exec jekyll post "My New Post"
To edit: _posts/
